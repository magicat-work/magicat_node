# magicat_node

---

## 快速搭建 Hysteria2 节点

```bash
curl -Ls https://raw.githubusercontent.com/MagicatAI/magicat_node/main/install.sh | sudo bash
```

---

## 服务器要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Linux （Debian13.1+ / Ubuntu 24.04+） |
| 最低配置 | 1核1G |
| 权限 | root |
| 端口 | 开放 UDP 443 |
| 依赖 | `curl`、`openssl`（多数系统已预装） |

> **注意**：Hysteria2 基于 UDP 协议，请确保云服务商的安全组 / 防火墙已放行 **UDP 443**
